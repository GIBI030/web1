HTML이란 무엇인가?

hypertext markup language is the standard markup language. The html tag marks the boundaries of the document's contents. All HTML documents contain an html tag, a head tag, and a body tag; the head and body tags mark the head and body sections of the HTML document,respectively.

Inside the head are the title, metadata, styles, and scripts. Generally, the head tag contains document information that is not rendered in the page, while the body contains the actual data presentation.
